# Telegram-game-bot

RPG-игра прямо в Telegram: сражения с монстрами, кланы, рынок, бизнес и многое другое.

## Быстрый старт

### Без Docker

```bash
# 1. Установите зависимости
pip install -r requirements.txt

# 2. Создайте файл .env на основе шаблона
cp .env.example .env
# Откройте .env и укажите BOT_TOKEN и ADMIN_IDS

# 3. Запустите бота
python bot.py
```

### С Docker

```bash
# 1. Создайте файл .env на основе шаблона
cp .env.example .env
# Откройте .env и укажите BOT_TOKEN и ADMIN_IDS

# 2. Соберите и запустите контейнер
docker compose up -d
```

## Переменные окружения

| Переменная  | Описание                                      | По умолчанию         |
|-------------|-----------------------------------------------|----------------------|
| `BOT_TOKEN` | Токен бота от @BotFather                      | `YOUR_BOT_TOKEN_HERE`|
| `ADMIN_IDS` | ID администраторов через запятую              | `123456789`          |
| `DB_PATH`   | Путь к файлу базы данных SQLite               | `game.db`            |

## Возможности

- 🗺 10 локаций с уникальными монстрами и боссами
- ⚔️ Боевая система с критами и уклонением
- 🏰 Кланы и клановые здания
- 🛒 NPC-магазин и игровой рынок
- 💼 Бизнесы и недвижимость
- 🎉 События и промокоды
- 🛡 Снаряжение: оружие, броня, шлемы, ботинки, кольца, амулеты
